# tab-rename（タブ名変更）

## 何のためのスキルか

ターミナルのタブ名を、今やっている作業の内容に合わせた短い日本語タイトルに変更するスキル。
複数タブで並行作業するとき、どのタブで何をしているか一目で分かるようにする。

summon-experts の Phase 2（キャラ設定後）で自動的に呼ばれ、議題に合わせたタブ名が付く。

## やっていること

1. 会話コンテキストから作業内容を読み取り、日本語4〜10文字のタイトルを生成
2. 環境を自動判定（tmux / VS Code）して適切な方法でリネーム
3. tmux の場合はセッション名がデフォルト（数字連番）なら作業ディレクトリ名に自動変更

## 工夫したこと

- **環境自動判定**: tmux と VS Code で内部処理を切り替え。VS Code 内で tmux を使っている場合は tmux 側を優先する
