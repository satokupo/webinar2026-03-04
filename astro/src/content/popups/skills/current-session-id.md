# current-session-id（セッションID取得）

## 何のためのスキルか

現在の Claude Code セッションのIDを取得するスキル。
セッションIDはログ保存やプランファイルの命名に必要だが、直接取得するAPIが存在しないため、間接的な手法で特定する。

planmode-rules（プランファイルのID埋め込み）、save-chatlog（ログファイル命名）、context-handover（引き継ぎファイル命名）から呼ばれる共通ユーティリティ。

## やっていること

1. stamp.py でユニークなマーカー文字列を会話ログ（JSONL）に書き込む
2. find.py でJSONLファイルを検索し、そのマーカーが含まれるファイルを特定
3. ファイル名のベースネーム（UUID形式）がセッションID

## 工夫したこと

- **APIがないところを突破**: セッションIDを取得する正規の手段がないため、「マーカーを書き込む → ログから逆引き」というハック的だが確実な手法を採用
