# skill-design-guide（スキル設計ガイド）

## 何のためのスキルか

スキルの設計・構造チェック・新規作成を支援するナレッジ集。
Anthropic が作った Claude Code 用の skill-creator の存在に気づく前に、skill-creator の中身を参照してもっと軽量化させる形で作ったもの。実行エンジンは持たず、「設計知識の辞書」として機能する。

直列フローの中で直接呼ばれるわけではないが、スキル作成・改善時に暗黙的に参照される基盤ツール。

## やっていること

1. 公式ガイドライン（anthropic-guidelines）とユーザー固有ガイドライン（user-guidelines）の2層で設計基準を提供
2. 構造チェック時は「提案」として提示し、自動修正は行わない
3. skill-creator（反復改善ツール）の発動判定を内包し、必要な場合のみユーザーに提案

## 工夫したこと

- **参照ルーティング**: 用途に応じて読み込むファイルを切り替える。構造チェックなら2ファイル、ウェブ参照やトークン最適化の根拠が必要なら追加ファイルを読む、という段階的な設計
- **自己成長の仕組み**: スキル起動時に Claude Code の GitHub Releases を14日間隔でチェックし、スキル関連の変更があればユーザーにナレッジの更新を提案する。公式の仕様変更に追従してスキル自身が成長していく構造
