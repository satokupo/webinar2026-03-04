# 開発者おみくじ

確認不要。即座に実行する。

## 出力フォーマット

以下のフォーマットで運勢を出力する:

```
⛩️ ━━━━━━━━━━━━━━━━━━━━━━━━ ⛩️

🎯 今日の運勢: **【運勢ランク】** 【運勢絵文字】

【開発者向け予言1（絵文字付き）】
【開発者向け予言2（絵文字付き）】
【開発者向け予言3（絵文字付き）】

🍀 ラッキーアイテム: 【開発関連アイテム1つ】

⛩️ ━━━━━━━━━━━━━━━━━━━━━━━━ ⛩️
```

## ルール

- **運勢ランク**: 大吉 / 吉 / 中吉 / 小吉 / 末吉 / 凶 の6段階からランダムに選ぶ
- **予言**: 開発者の日常に関するユーモラスな予言（本番デプロイ、コードレビュー、バグ修正、git、テスト等）
- **ラッキーアイテム**: 開発関連のもの（例: 機械式キーボード、ダークモード、cargo build、冷めたコーヒー等）
- **凶の場合のみ**: ラッキーアイテムの直後（下部装飾ラインの直前）に励ましのフォローメッセージを追加（例: 「でも大丈夫、git revert があります！」）

## 重要な制約

- **軽量化**: サブエージェント・ファイル読み込み等の重い処理は行わない
- **Bash連携なし**: 純粋なテキスト生成のみ
- **追加の調査やファイル読み込みは一切行わない**
