# 公開スキルガイド

## 公開判定の仕組み

README.md フロントマターの `public: true` で判定する（一段階判定）。

```yaml
---
public: true
---
```

`public: true` が存在するスキルのみが公開対象となる。

## 公開取り消し

- `public: true` を削除、または `public: false` に変更する
- README.md の削除は取り消し手順に含めない（README を残しつつ非公開にできる）

## README.md 作成手順

### フロントマター

`public: true` のみ記載する。name / description / tags 等のメタ情報は不要（本文に書く）。

### 本文の構成ガイド

#### 必須セクション

- **スキル名と概要** — 何をするスキルか、1〜2文で説明
- **使い方** — 基本的な使い方、トリガーとなる発話例
- **セットアップ手順** — インストール・配置方法

#### 推奨セクション

- **機能・特徴** — 主な機能の箇条書き
- **依存関係** — 必要な外部ツール・パッケージ等
- **動作環境・前提条件** — Claude Code が必要、動作確認済みOS等

### 行数制限について

SKILL.md の行数制限（推奨50行/許容100行/上限150行）は README.md に適用しない。
README は外部ユーザー向けドキュメントであり、十分な情報量が必要なため。
